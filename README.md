# extrasensorialnodeio

Arduino -> Rpi -> node.js, sensing everything and emitting with the help of socket.io

## Requirements
### Hardware
* Arduino
* Photoresistor
* Some wires
* 100k resistor
* Rpi (or any other computer you would like to run it)

### Software
* node.js
    * socket.io
    * express.js
* redis-server
